実装された機能のドキュメントを生成または更新します。

対象ドキュメント：
- コード内ドキュメント（JSDoc/TSDoc）
- API仕様書（docs/api/）
- 機能説明（docs/features/）
- アーキテクチャドキュメント（docs/design/）
- README/CHANGELOG更新

Task tool を使って doc-generator エージェントを起動し、適切なドキュメントを生成・更新します。

生成されるドキュメントには以下が含まれます：
- 機能概要と要件
- 技術仕様と型定義
- 使用方法とコード例
- エラーハンドリング
- テスト情報
- 関連ドキュメントへのリンク
